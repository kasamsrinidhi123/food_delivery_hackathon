# Food Delivery Data Analysis

This project analyzes food delivery data using Python and Pandas.

## Datasets Used
- orders.csv – Order transaction details
- users.json – User information and membership details
- restaurants.sql – Restaurant data including cuisine and ratings

## Objectives
- Merge multiple datasets using common keys
- Analyze revenue, order trends, and user behavior
- Answer analytical questions based on the dataset

## Tools & Technologies
- Python
- Pandas
- SQLite
- Jupyter Notebook (Google Colab)

## Files
- food_delivery_analysis.ipynb – Main analysis notebook
- orders.csv
- users.json
- restaurants.sql
